# Update 1.0

## Release 1: Plug-and-Play Portable Launch

This is the first open-source release milestone for the project.

### Goals
- Make the SSD/USB startup flow easier to use
- Improve first-run experience for beginners
- Provide a clear terminal-first launch path
- Make the product feel dependable on first use

### Included improvements
- Plug-and-play launcher for Chromebook/Linux
- Improved startup scripts for mounted SSDs
- Better first-run guidance for starter stack setup
- Clearer terminal messaging and setup flow

### Open-source update policy
- Users are not forced to upgrade
- Older versions remain valid and usable
- Users can stay on any release and still use the project
- New updates are additive and optional
